Send WhatsApp notifications to your customers through OnSend.

== Description ==

OnSend is a WhatsApp API provider that enables you to send WhatsApp notifications to your customers. With this plugin, you can send notifications to your customers when a new WooCommerce order is placed or the WooCommerce order status changes.

== Installation ==

1. Log in to your WordPress admin.
2. Search for **OnSend**.
3. Install and activate the plugin.
4. Navigate to "Plugins" in the sidebar to access the plugin list page.
5. Find "OnSend" and click "Settings" to access the plugin settings page.
6. Follow the instructions provided and update the plugin settings.
